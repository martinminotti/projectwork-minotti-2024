** AVVIO DELL'APPLICATIVO **

Aprire il terminale e navigare nella directory in cui è presente il file docker-compose.yml.

Eseguire il seguente comando per avviare i container Docker:
docker compose up -d




** ACCESSO ALL'APPLICATIVO **

Una volta che i container sono stati avviati correttamente, aprire un browser web e accedere all'URL
localhost
